# Selzee Documentation

Selzee provides an AI-powered chatbot for e-commerce stores. This documentation covers the dashboard user guide and the API reference for managing chatbots, data sources, and analytics.

**Language:** [English](./documentation/en/) | [Slovenčina](./documentation/sk/)

## Documentation

- [Dashboard Overview](./documentation/en/overview.json) — summary of chatbot activity and conversation history
- [Conversations](./documentation/en/conversations.json) — viewing conversation details and providing feedback

- [Agent Configuration](./documentation/en/agent.json) — appearance, intelligence, and action settings
- [Analytics](./documentation/en/analytics.json) — engagement funnels, purchase impact, and conversation insights
